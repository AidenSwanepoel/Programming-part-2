# PROG5121POE NetBeans Project

This is a ready-to-import Maven NetBeans project for the IIE Programming 1A POE Part 1.

## Files
- `src/main/java/com/iie/prog5121poe/Login.java`
- `src/main/java/com/iie/prog5121poe/Main.java`
- `src/test/java/com/iie/prog5121poe/LoginTest.java`
- `pom.xml`

## Open in NetBeans
1. Open NetBeans.
2. Click **File > Open Project**.
3. Select the `PROG5121POE` folder.
4. Wait for Maven dependencies to load.

## Run the program
- Right-click project > **Run**
- Or run with Maven exec plugin.

## Run tests
- Right-click project > **Test**

## Notes
- Console application only.
- Unit tests use JUnit 5.
- Username, password, and cell number validation are implemented according to the POE screenshots provided.
